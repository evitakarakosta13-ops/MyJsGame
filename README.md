#JavaScript Game

##Description

This is a simple interactive browser game built using JavaScript, HTML, and CSS. The project focuses on practicing core JavaScript concepts such as DOM manipulation, event handling, and basic game logic.

##Live Demo

 https://evitakarakosta13-ops.github.io/MyJsGame/

##Technologies Used

* HTML5
* CSS
* JavaScript

##Features

* Interactive gameplay
* User input handling
* Dynamic updates using DOM manipulation
* Simple and clean user interface

##What I Learned

* How to manipulate the DOM using JavaScript
* Handling user events (clicks, actions)
* Structuring a small front-end project
* Writing clean and readable code

##How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/evitakarakosta13-ops/MyJsGame.git
   ```
2. Open the project folder
3. Open `index.html` in your browser
